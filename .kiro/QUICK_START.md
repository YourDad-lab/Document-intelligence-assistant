# 🚀 Quick Start Guide

## All Servers Running ✅

Your complete RAG system is live!

## 🌐 Access Points

### 1. RAG Q&A System
**URL**: http://localhost:3000
- Ask questions about your documents
- Get AI-powered answers
- Interactive globe background
- **No login required**

### 2. Analytics Dashboard
**URL**: http://localhost:5000
- **Login required** (any username/password works)
- Real-time query tracking
- Performance metrics
- Query logs

## 🔑 Login to Analytics

1. Go to: http://localhost:5000
2. Enter **any** username (e.g., `admin`)
3. Enter **any** password (e.g., `password`)
4. Click "Sign In"
5. View analytics dashboard!

## 📊 What Analytics Shows

- **Total queries** tracked
- **Confidence levels** (high/medium/low)
- **Response times** (latency)
- **Top queries** by frequency
- **Recent logs** (last 20 queries)
- **Dataset usage** distribution

## 🎯 Test It Now

   - Example: "What is artificial intelligence?"

## 🚪 Logout

Click the **"🚪 Logout"** button in the top-right corner of the analytics dashboard.

## 🔄 Auto-Refresh

Analytics updates automatically every **5 seconds** - no need to refresh!

## 📝 Quick Tips

- ✅ Any username/password combination works for login
- ✅ Analytics tracks every query automatically
- ✅ Session persists until you logout or close browser
- ✅ No database setup needed
- ✅ All data stored in memory (resets on restart)
