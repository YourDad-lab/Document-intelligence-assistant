# RAG Document Q&A Frontend

Modern 3D animated frontend for the RAG Document Q&A system.

## Features

- Clean white + blue gradient design
- 3D animated background with floating shapes
- Smooth transitions and animations
- Glassmorphism effects
- Responsive design
- Confidence badges (high/medium/low)
- Collapsible source citations
- Error handling with graceful UI

## Setup

1. Install dependencies:
\\\ash
npm install
\\\`n
2. Configure API endpoint (optional):
Create \.env.local\:
\\\`nNEXT_PUBLIC_API_URL=http://localhost:8000
\\\`n
3. Run development server:
\\\ash
npm run dev
\\\`n
4. Build for production:
\\\ash
npm run build
npm start
\\\`n
The app will be available at \http://localhost:3000\`n
## Tech Stack

- Next.js 14
- React 18
- TypeScript
- Framer Motion (animations)
- Three.js + React Three Fiber (3D graphics)
- Axios (API calls)

## Design System

- Primary: White base
- Accent: Blue gradients (#667eea, #764ba2, #4A90E2)
- Effects: Glassmorphism, soft shadows, 3D transforms
- Animations: Smooth, professional, non-distracting
